# 实验助手项目开发与部署说明

### Docker Desktop安装

#### **1. 开启WSL服务**

以管理员身份运行Windows PowerShell，输入wsl，检查是否本机开启wsl服务：

<img src="_assets/实验助手项目部署说明/media/image1.png" style="width:6in;height:2.50903in" />

若未开启，则需按照以下步骤进行配置：右键单击 Windows 按钮并选择 “应用和功能” → 选择相关设置下右侧的 “程序和功能” → 选择 “打开或关闭 Windows 功能” → 选择 “Hyper-V”，然后单击 “确定”（注意：安装完成后，系统会提示你重新启动计算机，**此步骤针对win10系统**）

#### **2. 下载Docker Desktop**

官方下载地址：<https://www.docker.com/products/docker-desktop/>

安装参考教程：<https://www.quanxiaoha.com/docker/windows-install-docker.html>

安装完成，打开 Windows PowerShell，分布输入 **docker --version** 和 **docker-compose –version**，出现版本号即安装成功

<img src="_assets/实验助手项目部署说明/media/image2.png" style="width:6in;height:1.29444in" />
