# MOEA/D + LSMOP 网页操作步骤

## 1. 启动算法服务

- 目标服务名：moead-lsmop-ls
- 启动命令：

powershell -ExecutionPolicy Bypass -File docs/cases/moo-moead-lsmop/scripts/start-alg-with-nacos.ps1

- 检查命令：

powershell -ExecutionPolicy Bypass -File docs/cases/moo-moead-lsmop/scripts/check-nacos-readiness.ps1 -ServiceName moead-lsmop-ls -AlgHttpBase http://localhost:18083

## 2. 新增问题实例

- 导入 problem_instances/ 下任一 JSON
- 类别建议：moo-lsmop

## 3. 新增算法

- 导入 `algorithm_instances/moead-lsmop-ls-default.json`
- 算法名称：moead-lsmop-ls
- 服务名：moead-lsmop-ls
- 关键参数建议：
  - nVars = 500
  - populationSize = 200
  - maxGenerations = 300
  - seed = 20260225

## 4. 创建并执行计划

- 至少配置 1 次运行（建议 3 次）
- 查看执行结果中的指标汇总与 run 明细
