# MOEA/D + LSMOP 字段与参数说明

## 服务名一致性

- 本案例服务名：moead-lsmop-ls
- Nacos 注册名、算法库服务名、计划内算法项必须一致

## 参数说明

- nVars (int)：决策变量维度（大规模建议 >= 500）
- populationSize (int)：种群规模
- maxGenerations (int)：迭代代数
- crossoverProbability (double)：交叉概率
- mutationProbability (double)：变异概率
- seed (long)：随机种子（用于复现实验）

## 指标关注

- 逼近能力：IGD、GD
- 多样性：HV、Spread、Spacing
- 算法比较：Coverage
