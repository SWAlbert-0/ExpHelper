# 网页操作步骤（Python NSGA-II 案例）

## 1. 导入问题实例

1. 打开“问题实例管理”。
2. 点击“JSON导入”，拖拽 `problem_instances/zdt1-py-caseA.json`。
3. 导入成功后在列表确认实例名 `zdt1-py-caseA`。

## 2. 导入算法

1. 打开“算法库管理”。
2. 点击“JSON导入”，拖拽 `algorithm_instances/py-nsga2-zdt1-ls-default.json`。
3. 确认算法行中：
   - 算法名称：`py-nsga2-zdt1-ls`
   - 服务名：`py-nsga2-zdt1-ls`
   - 运行时：`Python`

## 3. 上传源码并构建

1. 在算法行点击“源码”。
2. 先执行：
   `powershell -ExecutionPolicy Bypass -File docs/cases/moo-python-nsga2-zdt1/scripts/package-source-zip.ps1`
3. 在弹窗上传 `py-nsga2-zdt1-ls-source.zip`。
4. 点击“上传源码”，再点击“构建并启动”。
5. 等待状态到 `SUCCESS/CHECK_PASSED`。

## 4. 执行计划

1. 打开“执行计划管理 -> 执行向导”。
2. 选择问题实例 `zdt1-py-caseA` 与算法 `py-nsga2-zdt1-ls`。
3. 先点“执行检查”，通过后提交执行。
4. 查看执行日志，确认出现 `PLAN_DONE`。

## 5. 验收

- 执行结果弹窗应有 `HV / IGD+ / GD / Spread / Spacing`。
- 若缺失，优先检查返回中是否存在 `paretoPoint_*` 记录。
