# 版本3交付说明

## 1. 交付范围
- 后端：`exphlp/`（执行计划、算法管理、问题实例管理、通知能力、运行链路稳定性改进）
- 前端：`exphlp-front/`（执行计划页重构、日志/结果展示、删除一致性、交互与排版优化）
- 运维脚本：`scripts/`（运行检查、容器启动与维护脚本整合）
- 文档：`docs/dev/`、`docs/user/`、`docs/paper/`（配置清单、维护手册、用户指引、论文素材）

## 2. 版本3验收门禁（本地）
1. 后端测试：
   - `cd exphlp && mvn -q test`
2. 前端构建：
   - `cd exphlp-front && npm run build:prod`
3. 前端端到端：
   - `cd exphlp-front && npx playwright test`
4. 配置文档校验：
   - `powershell -ExecutionPolicy Bypass -File scripts/check-config-doc.ps1`

## 3. 进入版本4前的最小门槛
1. 版本3工作区无临时垃圾文件（尤其 Office 锁文件、调试产物）。
2. 版本3提交后工作区为干净状态，可追溯、可复现。
3. 版本4从版本3稳定快照切分支开展 Python 端新增开发。

## 4. 说明
- 本文档用于“版本3收口”与“版本4启动”的统一基线。
- 如后续新增配置项或启动链路，需同步更新 `docs/dev/配置清单.md` 与本说明。
