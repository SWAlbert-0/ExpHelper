# 文档总览

为降低查找成本，文档按用途分为三层：

- `docs/user/`：面向使用者的部署、入库、执行、排障手册。
- `docs/dev/`：面向开发者的接口、错误码、迭代记录与治理文档。
- `docs/archive/`：历史材料归档（旧报告、转换产物、原始文档等）。
- `docs/examples/`：可运行案例与示例算法（当前包含 NSGA-II + ZDT1）。

## 快速入口

- 用户先看：`docs/user/快速上手-从部署到执行.md`
- 开发与联调：`docs/dev/接口总览-v2.md`
- 错误码：`docs/dev/错误码映射-v2.md`
- 历史记录：`docs/dev/开发日志.md`
- 示例案例：`docs/examples/moo-nsga2-zdt1/README.md`
