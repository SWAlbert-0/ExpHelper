# 用户文档导航

本目录只保留 3 份主文档，按以下顺序阅读：

1. `快速上手-从部署到执行.md`  
用于首次使用，按步骤完成一次完整闭环：部署 -> 入库 -> 执行 -> 查看结果。

2. `算法服务接入与容器化指南.md`  
用于接入自定义算法服务，重点是源码上传构建、Nacos 注册与执行计划联通。

3. `常见问题与排障.md`  
用于定位高频故障：Nacos 无实例、构建超时、结果为空、邮件通知失败等。

维护约定：
- `docs/user` 不再新增并行说明文档（如“新版部署方法”“真实案例测试文档”）。
- 用户行为或页面流程变更时，必须同步更新上述 3 份主文档中的对应章节。
- 算法服务模板统一以 `docs/templates/README.md` 为入口，不在用户文档中维护模板副本。
